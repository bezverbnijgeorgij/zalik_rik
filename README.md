## 📝 Опис проєкту
Мобільний додаток реалізує повноцінну систему локальної авторизації з екранами входу, реєстрації та головною сторінкою користувача. 

### Реалізований функціонал:
* **Навігація:** Переходи між екранами реалізовані за допомогою `Stack Navigator` (`@react-navigation/native-stack`).
* **Валідація полів:** * Перевірка email на коректність формату (наявність символу `@`).
  * Перевірка пароля на мінімальну довжину (не менше 8 символів).
  * Обов'язкове підтвердження пароля на екрані реєстрації та перевірка їхнього збігу.
* **Відображення помилок:** Інтерактивне виведення текстових повідомлень про помилки валідації на екранах форм.
* **Локальне сховище (AsyncStorage):** Реалізовано збереження облікових даних користувача під час реєстрації. Доступ до головного екрана можливий виключно після успішної перевірки введених даних із даними у сховищі.
* **Захист маршрутів:** Використання методу `navigation.replace` унеможливлює повернення на екран авторизації за допомогою системної кнопки «Назад» після успішного входу.
